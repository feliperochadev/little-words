# UI Changes Planning

This folder contains planning artifacts for UI, UX, and Design System changes.

## When to use

Create a document here when the change involves:
- Visual design system updates (colors, typography, spacing, tokens)
- Component library additions or modifications to `src/components/UIComponents.tsx`
- Screen layout changes or new interaction patterns
- Accessibility improvements
- UX flow redesigns or navigation changes
- Theme or branding updates (`src/utils/theme.ts`)

## File naming

```
YYYY-MM-DD_NN-<slug>.md
```

Example: `2026-03-17_01-new-bottom-sheet-design.md`

## Template

Use `.agents/plan/design/DESIGN-TEMPLATE.md` as a starting point, extending it with:
- **Before / After** visual description or mockup reference
- **Affected components** list
- **Design tokens changed** (colors, spacing, font sizes)
- **Accessibility considerations**
- **Platform-specific notes** (Android primary target)
